# 碰撞波形预测模型
## 任务描述
- 训练数据由 VCS（Visual Crash Studio）数值仿真软件生成，其计算基于宏观元法（MEM）。碰撞场景为车辆与静止厚墙壁（墙壁可旋转）发生碰撞，主要接触发生在车前端与墙壁之间。
- 输入为三个碰撞工况标量参数：碰撞速度 impact_velocity、碰撞角度 impact_angle 和车墙重叠率 overlap，三者均为浮点数。
    - 车辆碰撞速度范围为 [25 km/h, 65 km/h]；碰撞角度范围为 [-45°, 45°]；重叠率范围为 (-1, -0.25] ∪ [0.25, 1]。
    - 碰撞角度的正负表示墙壁绕其中心点的旋转方向，逆时针为正，顺时针为负。
    - 重叠率表示车辆正面向墙的投影长度占车辆车宽的比例；若重叠部分包含主驾侧，则重叠率为正，否则为负。重叠率为 1 代表全宽碰撞。
- 输出为三个方向的碰撞波形。
    - 包括主驾侧 B 柱上一选定点的 x 方向线加速度曲线、y 方向线加速度曲线，以及乘员舱绕 z 轴角加速度曲线。
    - 每条波形都从原始仿真结果中按 1 ms 步长取点，共 150 个时间点（1 ms 到 150 ms）上的加速度值，并且都已经过滤波处理。
    - 注意：由于仿真初始阶段墙壁与车辆之间存在间隙（无法为每个 case 确定准确间隙），因此每条波形起始阶段的若干时间点，加速度值可能接近零。
    - X 方向预测最重要，且数据信噪比最高；Y 方向预测重要性次之，部分 case 的信噪比较高，部分 case 较低；Z 方向预测最难，也不太重要，绝大部分 case 的信噪比较低。目前 Z 方向波形预测结果未用于后续损伤预测模型，但波形预测模型仍暂时保留该输出。

## 训练所用 VCS 数据集中的真值波形
VCS 仿真得到的波形数据经过严格的 CFC 低通滤波，因此不存在非常高频的毛刺噪声，只保留了碰撞过程中的部分局部震荡。由于波形数据采集自车辆乘员舱 B 柱的某一点，滤波后仍然存在的局部小幅波动和震荡，可能是结构正常变形等因素造成的。

## 预测波形 VS 真值波形
- X 方向预测指标最好，其次是 Y 方向，Z 方向最差（不过 Z 方向预测效果现在已经无关紧要）。目前看来，信噪比与预测难度呈正相关。
- 目前单独训练好的碰撞波形预测模型精度尚可。对于最关键的纵向加速度（X 方向）波形，预测精度较高，波形趋势在大多数情况下都能较好吻合，主要误差在于局部细节：预测波形一般平滑性更高，虽然学到了真值波形的主要趋势，但缺乏真值波形中存在的局部震荡和小幅波动。所以看上去，真值波形相比预测波形更“毛刺”、像是“噪声”更大（此处“噪声”并非指测量噪声，而是指波形中存在的局部震荡和小幅波动）。
- 不过，对于实际的 MADYMO 物理碰撞仿真而言，输入波形中的毛刺和小幅局部震荡，对最终提取的损伤结果指标（HIC、Dmax 等）影响很小（但我不确定这一特性是否也能被数据驱动的神经网络复现）。碰撞波形预测模型对 Y 方向波形的预测精度差强人意，但在纵向加速度响应明显的角度碰撞场景下，预测效果不错，趋势预测尚可，例如车辆整体向某一侧移动。不过，在那些车辆纵向宏观响应理论上并不明显的场景下，Y 方向波形的预测精度（iso-rating 指标）会差一些；而这类场景中 Y 方向波形可能本来就以波动震荡为主，趋势不那么明确，并且 iso-rating 指标反映的是波形形态的相对误差，而非绝对数值误差。比如在低速碰撞场景下，iso-rating 指标可能不高，但实际波形的数值差异很小。
